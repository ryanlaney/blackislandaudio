Set Analyzer

Use this tool to analyze and compare pitch-class sets. Enter sets using note names or pitch-class integers. If you want to analyze contour, you'll need to tag on the octave number at the end of the note name.

Don't use the characters a, b, t, or e for pitch classes 10 or 11!

     Example: 0,1,5,6
     Example: C,Db,F,Gb
     Example: C4,Db5,F4,Gb3
